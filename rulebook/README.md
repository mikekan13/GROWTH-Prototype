# GRO.WTH Canonical Rulebook

This is the authoritative rulebook for GRO.WTH mechanics. When this
disagrees with any other source, this wins.

## Order of authority
1. Mike's verbal rulings (captured here with dates)
2. This rulebook
3. `GROWTH-DESIGN-TRUTH.md` (design philosophy; rulebook supersedes it on specifics)
4. `GRO.WTH Repository/` (lore + reference; not canonical on mechanics)

## Files
- `rulebook.md` — the canonical rules, organized by section
- `rulings.md` — dated log of Mike's decisions; append-only
