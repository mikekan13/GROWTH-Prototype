# Basic Resolution System

## Core Resolution Procedures

There are **two distinct procedures** based on whether the character has training in the relevant skill:

### Step 0: Declarations & Locks (Before Any Dice)
- **Watcher (GM)** declares DR and knowable modifiers
- **All boosts** from allies, Nectars, items, stances must be committed now
- After Skill Die is revealed, **only Effort wagering** can change

### Unskilled Check Procedure
1. **Wager Effort** (pay it from attribute pools)
2. **Roll Fate Die (FD)**
3. **Total = FD + flat modifiers + Effort** vs **DR**

### Skilled Check Procedure  
1. **Roll Skill Die (SD)** openly (everyone sees result)
2. **Wager Effort** after seeing SD result (pay it from pools)
3. **Roll Fate Die (FD)**
4. **Total = SD + FD + flat modifiers + Effort** vs **DR**

### Resolution Components
- **Fate Die (FD):** d8 default (d4, d6, d12, d20 from character build)
- **Skill Die (SD):** Based on skill level 1-20 (see [[Skill_Level_Progression]])
- **Effort:** Points wagered from governing attribute pools
- **Flat modifiers:** Attribute bonuses, environment, equipment (not aid/guide)

## Success Determination

**Simple Success System:**
- **Meet or exceed DR:** Success
- **One point under DR:** Failure

*Note: Critical success/failure effects handled by Terminal injection system*

## Difficulty Rating Scale

**No Hard Caps:** DRs scale with character capabilities and can reach extreme levels

### Rough Guidelines
- **1-12:** Early game difficulty range
- **13-25:** Mid-game challenges  
- **26-40:** End-game skilled character range (Level 20 skill + d20 Fate Die + effort)
- **41-60:** Enhanced characters with modifiers and augmentations
- **61-100+:** Magical challenges requiring mana augmentation to boost checks
- **100+:** Epic/divine level tasks

### Character Capability Ranges
- **Skilled Checks:** Soft cap ~40 (Level 20 skill + d20 Fate + effort), higher with modifiers
- **Unskilled Checks:** Soft cap ~20 (d20 + effort), higher with modifiers  
- **Magic-Enhanced:** No practical limit due to mana augmentation potential

## Effort System

**Effort:** Additional points wagered from attribute pools to boost check results

### Effort Timing (KEY DIFFERENCE)
- **Unskilled Checks:** Must wager Effort **before** rolling any dice (blind commitment)
- **Skilled Checks:** Wager Effort **after** seeing Skill Die result (informed decision) but before Fate Die Roll

### Effort Usage Rules
- Effort is **always spent** from attribute pools regardless of success/failure
- **Skilled Checks:** Generally no hard cap (practical limits from pool sizes)
- **Unskilled Checks:** GM determines which attributes can provide effort
- **Training Value:** Seeing Skill Die first lets players make informed Effort decisions

### Quick Odds Reference (FD = d8)
After seeing SD + modifiers, if you need Fate Die to roll:
- **≥3** → 75% chance
- **≥2** → 87.5% chance  
- **≥1** → 100% chance
