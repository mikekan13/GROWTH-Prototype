# Skill System Overview

## Design Philosophy: Context Over Categories

The GROWTH skill system prioritizes **contextual flexibility** over rigid categorization. Skills are created through freeform naming with mechanical precision delivered through natural language triggers in effects and enhancements.

### Core Principles
- **Complete Creative Freedom:** Players create any skill name that fits their character
- **Contextual Enhancement:** Nectars and other effects use natural language to define when they apply
- **Simple Base Mechanics:** All skills follow the same dice progression system
- **GM-Player Collaboration:** Agreements on contextual application ensure smooth gameplay

## Universal Skill Mechanics

### Skill Creation and Naming
**Freeform Creation:** Players can name skills anything that makes sense for their character  
**Examples:** "Sword Fighting," "Ancient History," "Stealth," "Persuasion," "Demon Lore," "Acrobatic Combat"

**No Required Categories:** Skills don't need to fit into predetermined categories  
**Character Flavor:** Skill names should reflect character background and specialization

### Dice Progression System
All skills follow the same progression from Level 1-20:

| Level Range | Die Type   | Max Effort                 | Description           |
| ----------- | ---------- | -------------------------- | --------------------- |
| 1-3         | Flat Bonus | Fate Die Max               | +1, +2, +3 to FD roll |
| 4-5         | d4         | Fate Die Max + Skill Level | Basic dice competency |
| 6-7         | d6         | Fate Die Max + Skill Level | Improved capability   |
| 8-11        | d8         | Fate Die Max + Skill Level | Professional level    |
| 12-19       | d12        | Fate Die Max + Skill Level | Master level          |
| 20          | d20        | Fate Die Max + Skill Level | Godlike mastery       |

### Effort Capacity
**Maximum Effort:** Current skill level + Fate Die maximum  
**Example:** Level 16 skill (d12) + d8 Fate Die = 22 maximum effort points  
**Purpose:** Higher skill levels enable more reliable outcomes through effort wagering

## Enhanced Effects Through Context

### Contextual Triggers
Enhancement effects use natural language to define when they apply rather than relying on rigid skill categories.

**Template Format:** "[Effect Name]: [Bonus] when [contextual trigger in plain English]"

### Effect Examples
**Combat Enhancement:**
- Battle Fury: +2 to Skill Roll when making weapon attacks
- Flurry Strike: Make an additional attack per action when using melee weapons

**Social Enhancement:**
- Heavenly Beauty: +3 when using physical attractiveness to influence others
- Silver Tongue: You may reroll failed attempts one time to deceive or persuade

**Knowledge Enhancement:**
- Scholar's Focus: Reroll failed attempts to recall information
- Ancient Wisdom: +2 to Skill Roll when dealing with historical or mystical knowledge

**Physical Enhancement:**
- Nimble Fingers: Double dice outcome on skills involving manual dexterity
- Cat-like Reflexes: +3 when reacting to sudden dangers

## Integration with Other Systems

### [[Nectars_and_Thorns_System|Nectars]] and Skill Enhancement
**Primary Delivery Method:** Most advanced skill mechanics come through Nectars  
**Examples:**
- Multiple Actions: "Whirlwind Strike" nectar grants multiple attacks per action
- Specialized Techniques: "Master Lockpicker" nectar grants advantage on security systems
- Synergy Effects: "Battle Mage" nectar combines combat and magic skills

### [[Three_Pillar_Attributes]] Integration
**Skill Resolution:** Skills combine with appropriate attributes for task resolution  
**Attribute Selection:** GM and player agree on which attribute applies to each skill use  
**Contextual Application:** Same skill might use different attributes in different situations

### Character Creation Integration
**[[Seeds_Roots_Branches_System]] Foundation:** Starting skills emerge from character background  
**Narrative Consistency:** Skill names and levels should reflect character history  
**Growth Potential:** Characters can develop new skills throughout play

## Advanced Mechanics

### Combat Skills
**Combat Flagging:** Some skills are designated as combat skills during creation  
**Special Mechanics:** Combat skills may have additional rules for multiple attacks or special maneuvers  
**Enhancement Path:** Combat effectiveness increases through skill levels and combat-specific Nectars

### Skill Synergy
**Complementary Skills:** Related skills can provide bonuses to each other's use  
**GM Discretion:** Bonuses apply when skills logically support each other  
**Narrative Integration:** Synergy creates richer character specialization stories

### Skill Evolution
**Name Changes:** Skills can evolve and be renamed as characters develop  
**Specialization:** Broad skills can be refined into more specific expertise  
**Mastery Paths:** Level 20 skills may unlock unique character capabilities

## GM Guidelines

### Skill Application
**Collaborative Determination:** GM and player work together to determine when skills apply  
**Consistent Rulings:** Establish precedents for similar situations  
**Contextual Flexibility:** Allow creative skill applications that fit the narrative

### Effect Integration
**Natural Language:** Use plain English to describe when enhancements apply  
**Avoid Rigidity:** Don't require exact skill names for effect triggers  
**Player Agency:** Let players suggest how their skills might apply to situations

### Balance Considerations
**Effort Limits:** Maximum effort prevents skill levels from becoming overpowered  
**Contextual Restrictions:** Enhancement effects should have clear limitations  
**Narrative Grounding:** Require logical connections between skills and their applications

## System Design Strengths
**Creative Freedom:** Players can create any skill concept that fits their character  
**Mechanical Precision:** Effects use clear contextual language for consistency  
**Narrative Integration:** Skills naturally flow from and enhance character stories  
**Scalable Complexity:** Base system is simple, complexity comes through optional enhancements  
**Cross-System Synergy:** Skills integrate with magic, combat, and social systems
