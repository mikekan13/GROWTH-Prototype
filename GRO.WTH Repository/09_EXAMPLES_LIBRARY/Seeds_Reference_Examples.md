# Seeds Reference Examples

Reference tables of specific Seeds with statistical mechanics, providing concrete examples for character creation. These are pre-designed examples that GMs can use as-is or modify according to their karma budget and campaign needs.

## Statistical Balance Notes

All Seeds include **KV (Karma Value)** costs representing their power level and the karma investment required from GMs to offer them to players. Higher KV Seeds have more powerful abilities but cost more to implement.

## GM Implementation Notes

### Karma Value System
- **Low KV (70-90):** Standard options available to most GMs
- **Medium KV (100-150):** Requires established GM with karma reserves  
- **High KV (150+):** Premium options for experienced GMs with high karma investment

### Customization Guidelines
- GMs may modify these Seeds based on campaign needs
- KV costs should be adjusted if significant changes are made
- Balance considerations: powerful abilities should have corresponding limitations
- Seeds can be restricted or modified based on campaign setting

### Integration with Current System
- These examples complement the custom creation philosophy
- Can be used as templates for creating campaign-specific variations
- All mechanical values subject to GM discretion and campaign requirements
- Frequency budgets determined by individual GM karma allocation
