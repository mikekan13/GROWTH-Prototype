# KRMA System

The **KRMA** (Karma) system serves as GROWTH's **meta-currency** for character advancement, death recovery, and extraordinary achievements.

## KRMA Earning Rate

**Weekly Sessions:** Approximately **156 Karma per year**  
**Base Rate:** ~3 Karma per session average  
**Variable Factors:** Exceptional roleplay, major story achievements, creative problem solving  
**GM Discretion:** Additional awards for outstanding contributions

## Character Creation KRMA

**Starting Recommendation:** ~50 KV (Karma Value) for new characters  
**Base Template:** 200 KV total budget
- **100 KV:** Seed selection
- **100 KV:** Root selection  
- **Age Modifier:** -2 KV per year of starting age
**Customization:** Players can adjust within GM guidelines

## KRMA Expenditure Categories

### Reroll Opportunities
- **Reroll Death Save:** 2/5/10 Karma (escalating cost)
- **Reroll Critical Failure:** Variable cost
- **Reroll Important Check:** GM discretion

### Character Advancement  
- **Language Learning:** 3/5/8 Karma (by difficulty)
- **Skill Proficiency:** 10/14/19 Karma (by level)
- **Ability Score +2:** 50 Karma (once only per attribute)

### Special Purchases
- **Equipment Access:** Rare or unique items
- **Story Influence:** Minor narrative control  
- **Divine Favor:** Temporary blessing or protection

## KV (Karma Value) vs KRMA

**KV (Karma Value):** Character's total accumulated life experience  
**KRMA:** Spendable meta-currency for improvements  
**Relationship:** KV represents character's spiritual weight, KRMA represents available advancement potential

### KV Applications  
- **Character Power Level:** Higher KV = more powerful character
- **Mana Calculations:** Spirit mana = KV/10 rounded up
- **Death Mechanics:** KV affects resurrection difficulty
- **Social Recognition:** NPCs may sense character's KV

## KRMA and Death

### Death Recovery Costs
**First Death:** Standard KRMA cost for death save rerolls  
**Repeated Deaths:** Escalating costs with each subsequent death  
**Permanent Death:** Some deaths cannot be reversed with KRMA

### Body-KRMA Return  
**Death Mechanics:** Upon permanent character death, body's accumulated experience returns to GM as KRMA pool  
**GM Resource:** Allows GM to enhance NPCs, create special encounters, reward exceptional play  
**Balance Mechanism:** Ensures KRMA circulation in campaign

## [[Soul_Package_System]] Integration

**Soul Package Retention:** Character's Soul Package (Half Soul + all Spirit attributes) persists after death  
**Reincarnation Mechanics:** KRMA can be spent to use Soul Packages for new characters  
**Experience Transfer:** Allows some character development to carry between incarnations

## Campaign Integration

### Long-term Planning
**Character Arcs:** KRMA enables extended character development across multiple campaigns  
**Group Resources:** Parties can pool KRMA for major objectives  
**Legacy Effects:** KRMA expenditures can have lasting campaign consequences

### GM Tools
**Reward Structure:** KRMA provides clear advancement incentives  
**Balance Control:** GM controls KRMA flow to manage campaign power level  
**Story Integration:** KRMA expenditures can drive narrative developments
