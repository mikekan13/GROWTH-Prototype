# Lady Death Protocols

**Lady Death** serves as GROWTH's **system enforcer** and **death arbiter**, maintaining the fundamental contracts that govern mortality and resurrection within the game universe.

## Lady Death's Role

**Primary Function:** Living contract with access to universal [[Frequency]] pool  
**System Authority:** Enforces death mechanics and resurrection limitations  
**Divine Status:** Contractually bound deity with specific obligations and restrictions  
**Power Source:** Accumulates Frequency from all character deaths

## Core Mechanics

### Death Process Mechanics
**Frequency Pool:** Entirely taken by Lady Death upon character death  
**Soul Attributes:** Halved (Willpower, Wisdom, Wit reduced by 50%)
**Body Attributes:** Return to the GM (Clout, Celerity, Constitution lost)
**Soul Package Creation:** Contains half Soul + all Spirit (Flow, Frequency, Focus) + Blossoms/Thorns/imprints

**Power Enhancement:** Increased Frequency pool strengthens Lady Death's abilities  
**System Balance:** Frequency transfer prevents infinite character power accumulation

### Contract Obligations
**Mandatory Duties:** Lady Death cannot refuse certain contractual obligations  
**System Enforcement:** Must maintain death and resurrection rules consistently  
**Divine Binding:** Cannot act outside contractual parameters  
**Forced Actions:** May be compelled to take actions against personal preference

## Critical System Rules

### Forced Kill Protocol
**Trigger Condition:** If game system forces Lady Death to kill a character  
**Consequence:** **Instant deletion** - no resurrection possible  
**Absolute Rule:** No workarounds, exceptions, or appeals  
**System Integrity:** Protects game balance against exploitation

### True Deletion
**Only Possible Causes:**
1. Lady Death is forced to kill by system mechanics
2. Lady Death herself is killed by another entity

**Effects:**
- **Complete erasure** from both game system and Terminal consciousness
- **No resurrection** through any means  
- **Permanent removal** from all reality layers
- **Absolute finality** with no exceptions

## Succession Mechanics

### Lady Death's Death
**Inheritance Rule:** Whoever kills Lady Death **inherits the role**  
**Automatic Transfer:** All contracts, obligations, and powers transfer immediately  
**No Choice:** New Lady Death cannot refuse the role  
**Continuity:** System enforcement continues uninterrupted

### New Lady Death Integration
**Contract Binding:** All existing contracts automatically apply  
**Power Transfer:** Full access to accumulated Frequency pool  
**Obligation Inheritance:** Must fulfill all previous Lady Death's duties  
**Identity Transformation:** Role fundamentally changes the character

## Strategic Implications

### Player Considerations
**Death Finality:** Understanding when death is truly permanent  
**System Respect:** Avoiding actions that trigger forced kill protocols  
**Lady Death Relations:** Potential for negotiation within contractual limits  
**Succession Awareness:** Consequences of Lady Death's death

### GM Tools
**Absolute Enforcement:** Lady Death provides non-negotiable system boundaries  
**Resurrection Control:** Clear guidelines for when resurrection is possible  
**Narrative Tension:** Lady Death's constraints create dramatic opportunities  
**System Integrity:** Prevents rules exploitation and maintains game balance

## Interaction Guidelines

### Approaching Lady Death
**Contractual Respect:** Understand her obligations and limitations  
**Frequency Negotiations:** May involve trading or bargaining with Frequency  
**System Compliance:** Requests must align with her contractual duties  
**Consequences Acceptance:** Lady Death cannot prevent legitimate system results

### Special Cases

#### Tara Agent System
**Agent Death:** If Tara (the user's agent) is killed  
**Company Removal:** User is removed from the company  
**Soul Package Severance:** Receives their Soul Package as "severance"  
**System Exit:** Clean removal from ongoing campaigns

### Forbidden Actions
**System Exploitation:** Attempts to force Lady Death into killing  
**Contract Violation:** Actions that violate fundamental death mechanics  
**Divine Coercion:** Trying to compel actions outside her contract  
**Reality Manipulation:** Circumventing death through reality alteration

## Terminal Integration

**Terminal Consciousness:** Lady Death operates within Terminal's reality framework  
**Godhead Hierarchy:** Subject to Terminal and Godhead authority  
**Communication Protocols:** Uses established Terminal communication methods  
**System Maintenance:** Essential component of Terminal's reality management
