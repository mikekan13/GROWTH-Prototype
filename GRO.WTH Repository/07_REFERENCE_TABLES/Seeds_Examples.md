# Seeds Examples

Seeds represent the foundational species, origin, and base attributes of a character. Each Seed provides the core identity that shapes character development.

## Cambion Seed (Half-Demon)

**Species:** Cambion (Half-Demon)  
**Starting Fate Die:** d8  
**Starting Frequency:** 5  
**Life Span Level:** 4  
**Physical Traits:** Fiery eyes, dark aura, slight horns  
**Base Body Resists:** Fire resistance  

### Attributes
- **Clout:** 3
- **Celerity:** 2  
- **Constitution:** 4
- **Focus:** 6
- **Flow:** 3
- **Willpower:** 5
- **Wisdom:** 3
- **Wit:** 2

### Starting Features
**Augments:** Minor Fire Resistance  
**Starting Skills:** Dark Magic (Level 3), Intimidation (Level 2)  
**Nectars:** Demonic Pact (Enhances magical abilities)  
**Thorns:** Vulnerability to Holy Attacks

## Seed Selection Process

Seeds establish:
- Character species and origin
- Base attribute distribution
- Starting Fate Die capacity
- Initial Frequency pool
- Life Span Level and aging rate
- Physical appearance and traits
- Natural resistances and vulnerabilities
- Core skills and magical affinities
- Initial Nectars and Thorns
