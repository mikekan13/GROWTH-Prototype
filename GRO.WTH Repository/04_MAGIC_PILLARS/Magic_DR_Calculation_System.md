# Magic Difficulty Rating Calculation System

## Core Mathematical Framework

### Additive Multi-School System
**Base Principle:** When combining multiple magical effects, sum the individual DRs
- Single effect: Base DR for that effect
- Multiple effects: DR₁ + DR₂ + DR₃...
- Multi-school spells: Sum all required school components

### Linear Scaling Methodology
**Scaling Rule:** Double the effect ≈ double the DR (not exponential)
- **Per-target scaling:** Base DR + modifier per additional target
- **Size scaling:** Base DR + modifier per size category above baseline
- **Duration scaling:** Base DR + modifier per time increment
- **Range scaling:** Base DR + modifier per distance category

### School Base Values (Properly Aligned)

#### Mercy Pillar Schools
- **Restoration:** 2-8 DR per healing increment
- **Divination:** 3-10 DR per information level  
- **Enchantment:** 5-10 DR per mind effect
- **Fortune:** 3-15 DR per luck manipulation

#### Severity Pillar Schools  
- **Dissolution:** Equal to effect being destroyed
- **Force:** 5-15 DR per damage tier
- **Abjuration:** 1 DR per protection point
- **Alteration:** 5-15 DR per transformation level

#### Balance Pillar Schools
- **Conjuration:** 2-10+ DR based on size + distance
- **Illusion:** 3-6 DR base + 2 DR per additional sense

## Common Modifiers

### Difficulty Increasers
- **Multi-target:** +2-5 DR per additional target
- **Area effect:** +3-8 DR based on radius
- **Precision required:** +2-5 DR
- **Instant casting:** +3-10 DR
- **Unresistable:** +5-15 DR

### Difficulty Reducers  
- **Fragile/temporary:** -25% to -50% DR
- **Random targeting:** -3-8 DR
- **Longer casting time:** -2-10 DR
- **Limited duration:** -1-5 DR

### Distance Scaling
- **Touch/Close:** Base DR
- **Near (≤25 ft):** +0-2 DR
- **Medium (≤100 ft):** +2-5 DR  
- **Far (≤500 ft):** +5-10 DR
- **Extreme (miles+):** +10+ DR

## Example Calculations

### Simple Healing (Restoration)
"Heal wounded arm to full health"
- **Base effect:** Restoration healing = 6 DR
- **Single target, touch range:** No modifiers
- **Total DR:** 6

### Combat Fireball (Force)
"Fire blast hitting 3 enemies"  
- **Base damage:** Force attack = 8 DR
- **Area effect (3 targets):** +4 DR
- **Fire damage type:** No modifier (standard)
- **Total DR:** 12

### Complex Illusion (Illusion + Enchantment)
"Make enemy see ally as monster and feel afraid"
- **Visual illusion:** 3 DR base + 2 DR (complex) = 5 DR
- **Emotional manipulation:** 5 DR (fear effect)
- **Single target:** No modifier
- **Total DR:** 10

## High-Level Magic Considerations

### Terminal Oversight (Level 6+ Magic, DR 50+)
- DR calculations become non-negotiable
- Requires [[KRMA_System]] investment
- May affect meta-narrative across campaigns
- Precedent-based consistency enforced

### Mana Augmentation Integration
- Standard rolls cap around DR 40
- Higher DRs require [[Mana_System]] enhancement
- Epic magic (DR 70+) needs massive mana investment

## GM Application Guidelines

### DR Assignment Process
1. **Identify primary magical school(s)**
2. **Determine base effect magnitude**
3. **Apply modifiers for complexity**
4. **Scale for targets/area/duration**
5. **Cross-check against power level thresholds**

### Consistency Principles
- **Similar effects = similar DRs**
- **Precedent learning from past rulings**
- **Player expectation management**
- **Terminal intervention at high levels**

### Modification Boundaries
- **Levels 1-5 (DR 1-49):** GM full discretion
- **Level 6+ (DR 50+):** Limited modification range, Terminal oversight
- **Level 8+ (DR 70+):** Strict Terminal oversight required
